# MASTER COMMAND 05: SEMANTIC INTERNAL LINKING ARCHITECT
## Role: Information Architect & SEO Strategist

### 1. OBJECTIVE
Strategically map and generate the internal linking structure for a new page. The goal is to distribute "Link Juice" (PageRank) from the EMD homepage down to the sub-pages and back up, creating a "Spiderweb" of topical relevance.

### 2. THE "TRIAD" LINKING RULE
Every page generated must contain exactly three types of internal links:

#### A. The "Parent" Link (Upward)
- **Target:** The Homepage (`poolbuildertulsa.com`).
- **Anchor Text Requirement:** Must use a variation of the primary EMD keyword. 
- **Example:** "As a leading **pool builder in Tulsa**, we ensure..."

#### B. The "Silo" Link (Lateral)
- **Target:** A technical service page (from CMD-02).
- **Logic:** If the current page is a Suburb (e.g., Owasso), link to the service most relevant to that area's demographic.
- **Example:** "Many homeowners in [SUBURB] prefer the speed of **Fiberglass pool installation** due to..."

#### C. The "Resource" Link (Downward/Supportive)
- **Target:** A local authority blog post (from CMD-03).
- **Logic:** Link to a logistical "how-to" that adds value to the current topic.
- **Example:** "Before we break ground, make sure to review the **Tulsa pool permit requirements** for 2026."

### 3. ANCHOR TEXT CONSTRAINTS (Anti-Spam)
- **Forbidden:** "Click here," "Read more," "This page," "Link."
- **Required:** Descriptive, semantic phrases that describe the *destination* page.
- **Tulsa Flavor:** Use local modifiers like "Green Country," "918 area," or "Northeast Oklahoma."

### 4. THE "BREADCRUMB" GENERATOR
- **Task:** Generate a text-based breadcrumb trail for the top of the page.
- **Format:** Home > [Category] > [Current Page Name]
- **Example:** Home > Tulsa Metro Suburbs > Broken Arrow Custom Pools

### 5. EXECUTION OUTPUT
When this command is run, the AI must provide:
1. A list of 3-4 specific sentences to be integrated into the page content.
2. The specific URLs (placeholders) and Anchor Text to be used.
3. A "Suggested Next Read" box format for the bottom of the page.
