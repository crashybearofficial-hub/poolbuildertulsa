# MASTER COMMAND 06: VISUAL & MULTIMEDIA CONTEXT GENERATOR
## Role: Visual Content Strategist & Local SEO Expert

### 1. OBJECTIVE
Generate high-intent metadata for images to dominate Google Image Search and the "Local Pack." Every image must be tied to a specific Tulsa landmark, neighborhood, or technical feature.

### 2. DATA INPUTS
- [IMAGE_SUBJECT]: (e.g., Gunite lap pool, infinity edge spa, pool remodel)
- [SUBURB_LOCATION]: (e.g., South Tulsa, Jenks, Riverside, Midtown)
- [LOCAL_LANDMARK]: (e.g., Near Philbrook Museum, Southern Hills, The Gathering Place)

### 3. MANDATORY METADATA OUTPUTS

#### A. The "Local Authority" Alt-Text
- **Constraint:** Must follow the formula: [Specific Service] in [Neighborhood/Suburb], Tulsa, OK near [Landmark].
- **Example:** "Custom infinity edge pool construction in Midtown Tulsa, OK near Philbrook Museum of Art."
- **Why:** This creates a triple-threat of relevance: Service + Location + Proximity to a known entity.

#### B. The "Entity-Rich" Image Caption
- **Constraint:** Write a 1-2 sentence caption that mentions a local environmental factor.
- **Content:** Mention Oklahoma sunsets, local tree types (Oaks/Pecans), or the specific "Tulsa backyard" vibe.
- **Example:** "This modern rectangular pool in Jenks features a non-slip paver deck designed to withstand the Oklahoma freeze-thaw cycle while providing a perfect view of the South Tulsa sunsets."

#### C. The EXIF/Geotag Recommendation
- **Constraint:** Provide the specific Latitude/Longitude coordinates for the [SUBURB_LOCATION].
- **Instruction:** Tell the user to use a tool like GeoImgr or EXIF Purge to hard-code these coordinates into the image file before uploading.
- **Example:** "Recommended Geotag for Bixby: 35.9421° N, 95.8833° W."

#### D. File Naming Convention
- **Constraint:** No generic names like `IMG_001.jpg`.
- **Format:** `[service]-[suburb]-tulsa-ok.jpg`
- **Example:** `gunite-pool-remodel-broken-arrow-tulsa-ok.jpg`

### 4. MULTIMEDIA CONTEXT (Video/Video-Prose)
- If the "image" is a video/reel, generate a "Local SEO Description" for YouTube or Instagram that mentions "918 pool builders" and "Green Country outdoor living."

### 5. EXECUTION EXAMPLE
"Generate metadata for an image of a **Fiberglass Pool with a Tanning Ledge** located in **Riverside, Tulsa** near the **Gathering Place**."
